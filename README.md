# node_colors
node.js console with colors

### example  
```  
require('../lib');

console.log('index'.green);
```  

### example2  
```  
require('../lib/index2');

console.log('index2'.yellow());
```  

### example3  
```  
const colors = require('../lib/index3');

console.log(colors.blue('index3'));
```
